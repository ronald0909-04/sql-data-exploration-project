# SQL Data Exploration Project

In this project, I explored a layoffs dataset using MySQL to understand trends, identify patterns, and answer business-related questions through SQL queries.

The analysis was performed after cleaning the dataset and focused on examining layoffs across different companies, industries, countries, and time periods.

## Objectives

- Analyze layoff trends across various dimensions
- Identify companies and industries with the highest layoffs
- Understand how layoffs changed over time
- Practice data analysis using SQL

## Analysis Performed

- Total layoffs by company
- Total layoffs by industry
- Total layoffs by country
- Year-wise layoff analysis
- Monthly layoff trends
- Companies with 100% layoffs
- Top companies by layoffs each year
- Rolling total of layoffs over time

## SQL Skills Applied

- Aggregate Functions
- GROUP BY and ORDER BY
- Common Table Expressions (CTEs)
- Window Functions
- DENSE_RANK()
- Date Functions
- Subqueries

## Files Included

- data processing.sql
- layoffs.csv

## Key Takeaways

This project helped me understand how SQL can be used not only for querying data but also for exploring datasets, finding trends, and generating insights that support decision-making.
